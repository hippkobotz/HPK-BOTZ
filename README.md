## Clone this project

```bash
> git clone https://github.com/hippkobotz/HPK-BOTZ
```

```bash
>Command BOT $
```

## Instale as dependências:
Antes de executar o comando abaixo, certifique-se de estar no diretório do projeto que
você acabou de clonar !!

```bash
> cd HPK-BOTZ
> bash install.sh
```

### Usage
```bash
> npm start
```
